# Hexo 页面设计要求

## 1. 项目说明

- 当前项目是基于 Hexo 工具 构建的展示性网站, 主题使用 butterfly
- 网站的主题是:
  - 提供多款 AI 软件产品的展示和推广 
  - 提供基于 软件产品的技术博客或使用说明

- 网站布局及配色: 简洁美观，充满科技范



## 2. 导航分类

整个页面分成3类:

- 主页 - 使用支持左右滚动的 Gallery 布局, 每个Gallery 中间增加一个 '点击查看详情' 按钮, 链接到一个软件产品的详情展示页面
- 导航菜单 ： 软件产品 - 每个软件作为一个子菜单，对应软件产品文章， 点击后跳转到软件产品的详情展示页面
- 博客 菜单 : 对应存放博客或说明类文章
